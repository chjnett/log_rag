# CLI-Mate 디자인 시스템

## 컨셉: "Terminal Noir"

개발자 도구답게 **다크 터미널 감성** + **모던 미니멀리즘**을 결합한 디자인

### 핵심 특징

| 요소 | 설명 |
|------|------|
| **컬러** | 순수 블랙 베이스 + 네온 악센트 (Green/Cyan) |
| **타이포** | 모노스페이스 폰트 (JetBrains Mono / Fira Code) |
| **레이아웃** | 터미널 창 스타일 카드, 글로우 효과 |
| **인터랙션** | 타이핑 애니메이션, 깜빡이는 커서 |

---

## 컬러 팔레트

```
Background:  #0a0a0a (거의 순수 블랙)
Surface:     #121212 (카드 배경)
Border:      #1e1e1e (미묘한 경계)

Primary:     #00ff88 (네온 그린 - 성공/해결책)
Secondary:   #00d4ff (시안 - 링크/액션)
Error:       #ff4757 (레드 - 에러 강조)
Warning:     #ffa502 (오렌지 - 경고)
Muted:       #6b7280 (회색 - 보조 텍스트)
```

- 유사 사례는 유사도 % 바 표시

## UI 컴포넌트 스타일

### 터미널 윈도우 카드

```css
/* 터미널 창 느낌 */
- 상단 바: ●●● (빨강/노랑/초록 버튼)
- 둥근 모서리 (radius: 8px)
- 미세한 그림자 + 테두리 글로우
- 호버 시 border-color 변화
```

### 태그 뱃지

```css
/* 터미널 명령어 스타일 */
- 배경: 반투명 컬러
- 폰트: 모노스페이스
- 앞에 $ 또는 # 기호
```

### 버튼

```css
/* 미니멀 + 글로우 */
- Primary: 네온 그린 테두리 + 호버 시 글로우
- Ghost: 투명 배경 + 텍스트만
```
